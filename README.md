# How to use?

### 1. Installation
``` 
~$ npm i cprint
```

### 2. Connect cPrint
```
const cPrint = require('cprint');
```

### 3. Use!
```
cPrint( 'text', 'color' );
```

## Text
### A-Z, a-z, 0-9, ',', '.', ':', ';'
